This project is a test assignment implementing a simple Employee Management REST API using **.NET**, **ADO.NET**, **PostgreSQL**, and **.NET Aspire**.

## Implement two REST API endpoints:

### 1. GetEmployeeWithSubordinates
Returns an employee along with all their subordinates.

### 2. UpdateEnable
Updates the `Enable` flag for a specific employee.


## To launch project:
1. Clone repository.
2. Navigate to the project directory.
3. Make sure Docker Desktop is running.
4. Start the application:
```
dotnet run --project .\iConductTestTask.AppHost.csproj
```
5. Open the URL:
```
https://iconducttesttask.dev.localhost:17091
```
